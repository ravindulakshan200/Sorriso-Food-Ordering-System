# 🍕 Sorriso Food

A modern full-stack restaurant ordering and reservation platform built with Next.js, Supabase, and TypeScript.

## ✨ Features

- 🍽️ Digital Menu
- 🛒 Shopping Cart
- 💳 PayHere Payment Integration
- 📦 Order Tracking
- 📅 Table Reservations
- 🔐 Admin Dashboard
- 📊 Order Management
- 🌐 Multi-language Support
- 📱 Responsive Design

## 🚀 Installation

```bash
git clone <repository-url>

cd foooood/frontend

npm install

npm run dev
```

---

## 📁 Environment Variables

Create:

```
frontend/.env.local
```

Example:

```
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
PAYHERE_MERCHANT_ID=
PAYHERE_MERCHANT_SECRET=
```
